# Адмін-панель КОСМОДЕНТ

## Вхід до адмін-панелі

### Демо-режим (без бекенду)
- **Логін:** `admin`
- **Пароль:** `admin`

### З бекендом
1. Запустіть бекенд: `cd backend && npm run dev`
2. Заповніть базу даних: `npm run seed`
3. Використовуйте ті ж креденціали:
   - **Логін:** `admin` (або `admin@kosmodent.ua`)
   - **Пароль:** `admin`

## Функціонал

### Dashboard
- Перегляд статистики записів
- Кількість послуг та лікарів
- Відгуки пацієнтів

### Записи (Appointments)
- Перегляд всіх записів
- Фільтрація по статусу
- Зміна статусу запису

### Послуги (Services)
- Перегляд списку послуг
- Додавання/редагування/видалення
- Зміна порядку відображення
- Активація/деактивація послуг

### Лікарі (Doctors)
- Перегляд списку лікарів
- Додавання/редагування/видалення
- Активація/деактивація

### Відгуки (Reviews)
- Перегляд відгуків
- Схвалення/відхилення
- Видалення відгуків

### Блог (Blog)
- Перегляд статей
- Публікація/зняття з публікації
- Додавання/редагування/видалення

### Налаштування (Settings)
- Контактна інформація
- Графік роботи
- Соціальні мережі

## Адреси

- **Головна сторінка:** http://localhost:3000
- **Адмін-панель:** http://localhost:3000/admin/login
- **Dashboard:** http://localhost:3000/admin/dashboard
